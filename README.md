# FileSphere

FileSphere is a small browser-based file management app inspired by the original C++ console program in `ads.cpp`.

## Run

Open `index.html` in a browser. The app is fully static and stores files in `localStorage`.

## Features

- Create, edit, delete, and read files
- Search by name, extension, and content
- Filter common text, code, and data files
- Sort by name, size, update time, or extension
- Import a local file into the workspace
- Download the active file
